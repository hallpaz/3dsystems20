## Assignment 2 - Rendering a Scene with a Differentiable Renderer

The objective of this assignment is to render a scene using a differentiable renderer and exploit some capabilities of this system. We aim to render equirrectangular panoramas, building a scene with a single sphere parameterized by latitude and longitude.
